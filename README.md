# 🚀 CareerWise Assessment Platform

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https%3A%2F%2Fgithub.com%2Fsravanigorentla%2Ffsad-review2&env=MONGODB_URI,JWT_SECRET&project-name=careerwise&repository-name=fsad-review2)

Premium AI-powered career assessment platform with a glassmorphic UI, real-time analytics, and a comprehensive role-matching engine.

## 🛠️ Tech Stack
- **Frontend**: React 19, Vite, TypeScript, Lucide Icons
- **Backend**: Node.js, Vercel Serverless Functions
- **Database**: MongoDB Atlas
- **Styling**: Vanilla CSS (Premium Glassmorphism)

## ⚡ Features
- **10 Core Questions** (from a bank of 22) for accurate matching.
- **7 Detailed Career Roadmap** including AI, Cyber Security, and DevOps.
- **Live Timer** and progress tracking.
- **Admin Dashboard** with real-time stats and demo mode.

## 🚀 One-Click Deployment
Click the button above to deploy this project to Vercel instantly. You will need to provide your `MONGODB_URI` from MongoDB Atlas during the setup.

## 💻 Local Development
1. Clone the repo
2. Run `npm install`
3. Set your `.env` variables
4. Run `npm run dev`
